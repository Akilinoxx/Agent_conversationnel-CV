Intégration de l'Agent Conversationnel CV dans votre Portfolio

Instructions d'intégration

Voici comment intégrer l'agent conversationnel CV dans votre portfolio GitHub Pages :

Étape 1 : Ajouter le script d'intégration

Ajoutez le script suivant dans votre fichier HTML principal (par exemple, index.html) :

<script src="https://votre-api-deployee.com/static/portfolio-integration.js"></script>

Étape 2 : C'est tout !

Le script créera automatiquement un bouton de chat flottant dans le coin inférieur droit de votre portfolio. Aucune autre modification n'est nécessaire.

Personnalisation (optionnelle)

Si vous souhaitez personnaliser l'apparence ou le comportement du chatbot, vous pouvez modifier le fichier portfolio-integration.js selon vos besoins.

Déploiement de l'API

Pour que l'intégration fonctionne, vous devez déployer l'API sur une plateforme comme Render, Deta Space ou Heroku. Une fois déployée, mettez à jour l'URL de l'API dans le fichier portfolio-integration.js :

// Remplacez cette ligne
const API_URL = 'https://votre-api-deployee.com/api/chat/stream';

// Par l'URL réelle de votre API déployée
const API_URL = 'https://votre-api-reelle.com/api/chat/stream';
